핵심 기능
🗺️

스마트 경로 최적화

Google Routes API 기반으로 여러 장소를 최적 순서로 자동 정렬. 고정 시간 핀, 영업시간 체크, Before/After 비교까지.

차별화 기능
🔄

Plan B 대안 계획

메인 일정마다 대안 계획을 추가하세요. 비가 와도, 문 닫아도, 대안이 있으니 괜찮습니다.

💰

더치페이 정산

다중 통화 지원, 영수증 OCR 스캔, 최소 거래 횟수로 자동 정산. "누가 얼마 보내면 끝" 한눈에.

👥

실시간 협업

초대 링크 하나로 친구를 여행에 초대. 일정, 지출, 준비물을 함께 편집하고 실시간 동기화.

📍

장소 저장 & 컬렉션

가고 싶은 장소를 검색해서 저장하고, 컬렉션과 태그로 정리. 저장한 장소는 바로 일정에 추가.

📋

일정 관리

일차별 자동 그룹화, 15+ 카테고리, 장소 검색 연동. 체크박스로 여행 중 진행 상황을 추적하세요.

🧳

준비물 체크리스트

카테고리별 자동 정렬, 수량 입력, 진행률 시각화. 여행 전 빠뜨리는 것 없이 준비 완료.

📸

추억 & 피드

여행 중 사진과 메모로 추억을 기록하고, 여행을 피드로 발행해서 다른 여행자와 공유하세요.

📤

내보내기 & 공유

여행 일정을 PDF, HTML로 내보내기. 공유 카드를 자동 생성해서 카카오톡, 인스타로 바로 공유.
